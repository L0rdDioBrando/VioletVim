## Requirements:

Neovim 0.12+,

git,

tree-sitter CLI,

yazi (My config see [here](https://github.com/L0rdDioBrando/violet-dots/tree/main/modules/configs/yazi)),

clang *or* gcc,

fzf,

ueberzugpp,

rg,

fd,

Nerd Fonts (FiraCode in the screenshot)

---

## Install:

Make backup your configs:

```sh
mv ~/.config/nvim ~/.config/nvim.back
```
Install my dotfiles:

```sh
git clone https://github.com/L0rdDioBrando/VioletVim.git $HOME/.conifg/nvim
```
